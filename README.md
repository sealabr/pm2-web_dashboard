# pm2-web_dashboard
Web dashboard for pm2, using nodeJS with CoreUI template and socket.io

# Install PM2

$ npm install -g pm2 
$ npm install -g pm2-web

# run pm2 web_dashboard

$ pm2 web

# Run this ....

$ pm2 start server.js -- --port 4001

# run in browser

http://localhost:4001
